# nwjs-react-webpack-boilerplate
standalone web application boilerplate with nwjs

## install on local

```
$ npm run build

```

## how to build

```
$ npm run prod
$ /path/to/nwjc build/js/bundle.js build/app.bin
```

## how to run

```
$ npm run prod
$ /path/to/nwjs build
```
